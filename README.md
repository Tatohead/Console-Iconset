<sub>The objects may apppear slightly off compared to the physical object due to lack of references. Not all consoles/variants will be listed since they have yet to be made. Some are enlisted together due to running the same software or high compatibility. Phones are categorized in computers. I will not post them on other sites for personal reasons. However, feel free to use these icons in projects. All consoles are listed in the manufacturer's/developer's name.</sub>
